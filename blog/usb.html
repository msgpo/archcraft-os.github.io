Create A Bootable USB With Archcraft

dd (Linux)

If you're already using a Linux based OS, you can create an Archcraft bootable USB with dd. dd creates both BIOS and UEFI bootable USB & this method is recommended due to its simplicity and universal availability. Follow the steps below to create a bootable USB with dd:

  • Open terminal and type lsblk to get the USB info. In my case, it's /dev/sdb.
  • Run sudo dd bs=4M if=path/to/archcraft.iso of=/dev/sdX status=progress oflag=sync & enter your password.
  • Make sure you replace "path/to/archcraft.iso" with the Archcraft ISO path & "sdX" with your disk name from the above command.
  • This will create both BIOS & UEFI bootable USB for Archcraft.


Etcher (Linux, Mac, Windows)

Etcher is an OS image flasher built with Node.js and Electron, capable of flashing an SDCard or USB drive. It protects you from accidentally writing to your hard-drives and ensures every byte of data was written correctly. Follow the steps below to create a bootable USB with Etcher:

  • Launch Etcher. Click on Select Image & select the Archcraft ISO.
  • Select your USB drive, be careful to select the right one.
  • Click on Flash! & wait.
  • This will create both BIOS & UEFI bootable USB for Archcraft.


Other Options

Linux - CLI


  • Using cat: Run cat path/to/archcraft.iso > /dev/sdx with root permissions.
  • Using cp: Run cp path/to/archcraft.iso /dev/sdx with root permissions.
  • Using tee: Run tee < path/to/archcraft.iso > /dev/sdx with root permissions.

Windows - Rufus


Rufus is a multi-purpose USB ISO writer. It provides a graphical user interface and does not care if the drive is properly formatted or not. Simply select the Archcraft ISO, the USB drive you want to create the bootable Archcraft onto and click START.

If the USB drive does not boot properly using the default ISO Image mode, DD Image mode should be used instead.

  • For Rufus version < 3.0 select DD Image mode from the drop-down menu on the bottom.
