Install Archcraft On A UEFI System (With Encryption)

Follow the steps below to install Archcraft on a UEFI system (with encryption, which is optional for you) -

  1. After creating a UEFI bootable USB, boot system in UEFI mode with it & select "Archcraft 64-bit UEFI".

  2. On login screen, enter "liveuser" as the password & Login to openbox desktop.

  3. Press Window or Alt + F1 key to open launcher, Select "Install Archcraft" & launch installer.

  4. Now, Select the language & press ENTER. you'll see some messages like "All checks passed!"

  5. Each step must be followed in ORDER. Now, Select "Prepare Installation" & press ENTER.

  6. Select "Set Virtual Console" & press ENTER.

  7. I'm selecting "US". Choose whichever you prefer & press ENTER.

  8. Now, Select "Set Desktop Keyboard Layout" & press ENTER.

  9. Again, whichever you prefer. I'm selecting "US".

  10. Now, Select "List Devices (optional)" & press ENTER.

  11. Here, You can see a list of devices and partitions on your system. Remember the name of device or partition in which you want to install Archcraft.

  12. Now, go ahead and select "Partition Disk" & press ENTER.

  13. Select the device/disk you want to partition & press ENTER. I'm going to install archcraft on first HD, /dev/sda.

  14. Select "gparted" & press ENTER. You can also use "parted" if you prefer CLI.

  15. I'm installing it on an empty disk, so let's create a partiton table first. Select "gpt" & APPLY.

  16. Here, you can create "root", "boot", "home" and "swap" partitions. when done partitioning, write the changes & exit gparted.

  17. Here, i created a "boot" and a "root" partition. I need a separate "boot" partition for UEFI and Encryption.

  18. Now, Let's setup encryption. Select "LUKS Encryption" & press ENTER. if you don't want to use encryption, you can skip these steps & jump to Step 28 "Mount Partitions".

  19. I am selecting "Automatic LUKS Encryption". Again choise is yours, Select whichever you prefer & press ENTER.

  20. Select "root" partition & press ENTER. (In my case it's /dev/sda2)

  21. Set the name for encrypted partition. I choose the default.

  22. Enter the "un/encryption" password to un/encrypt partition & press ENTER.

  23. Re-enter the same "password" & press ENTER. Wait & ignore any warnings like /run/cryptsetup is missing.

  24. Now the partition is opened and ready for mounting. Press ENTER to Exit.

  25. Select "back" to exit this screen.

  26. If you want to use LVM, Select "Logical Volume Management" & press ENTER.

  27. Here you can setup LVM settings. I'm skipping this step as i don't want to use it.

  28. Now, Mount the partitions we've created. Select "Mount Partitions" & press ENTER.

  29. If you setup encryption, then choose "/dev/mapper/cryptroot" else just select "/dev/sda2" to mount as ROOT & press ENTER.

  30. Select "ext4" & press ENTER.

  31. Select "Yes" & press ENTER.

  32. Press Cancel and Ignore it. I mean it's totally on you if you what you're doing.

  33. You'll see Mount Successful! message when mounted successfully.

  34. Now, mount "swap" partition (if you've created any). I'm using a swapfile here.

  35. Enter the size of swap file, Must be equal to "System Memory (RAM)" & press ENTER.

  36. Now let's mount the "UEFI boot" partition. Select "/dev/sda1" & press ENTER.

  37. Select "/boot" & press ENTER.

  38. You'll see Mount Successful! message when mounted successfully.

  39. Select "Done" & press ENTER.

  40. Select "Back" & move to next step.

  41. Select "Install Base" & press ENTER.

  42. Select "Install Base Packages" & press ENTER. It will copy the rootfs to your root partition.

  43. Wait till it finished.

  44. Now, Select "Run Mkinitcpio" & press ENTER.

  45. It'll generate the initramfs and fallback initramfs images.

  46. Now, Select "Install Bootloader" & press ENTER.

  47. Select "grub" (recommended) & press ENTER. You can choose systemd-boot if you want.

  48. Press ENTER to set grub as default bootloader.

  49. Now, go back to next screen.

  50. Select "Configure Base" & press ENTER.

  51. Select "Generate FSTAB" & press ENTER.

  52. Select "UEFI Part UUID" & press ENTER.

  53. Select "Set Hostname" & press ENTER.

  54. Enter the hostname (default is archcraft) & press ENTER.

  55. Select "Set System Locale" & press ENTER.

  56. I'm Selecting "en_US.UTF-8". Choose the one you want & press ENTER.

  57. Select "Set Timezone and Clock" & press ENTER.

  58. Select your continent. Mine's "Asia".

  59. Select the City. Mine's "Kolkata"

  60. Select "Yes" & press ENTER.

  61. Use "localtime" if you're dualbooting with windows else, UTC.

  62. Now, Select "Set Root Password" & press ENTER.

  63. Enter the password for root account.

  64. Select "Add New User(s)" & press ENTER. You can create multiple user accounts here.

  65. Type your username, Must be lower case. This will be your regular user account.

  66. Enter password for new user.

  67. And a regular user will be created.

  68. Select "Security and systemd Tweaks" & press ENTER.

  69. Here, select and use whichever tweak you like. when you done, hit Back and go to next screen.

  70. Now, Select "Review Configuration Files" & press ENTER.

  71. Here's a list of all config file you can review and modify. For Ex - let's modify /etc/mkinitcpio.conf

  72. Here you can remove unnecessary modules. Let's say i've installed archcraft on an INTEL machine. So, i'll only keep i915 and remove everything else from the modules array.

  73. Also, I'll only keep the necessary HOOKS. Let's say I've installed archcraft without encryption. in this case plymouth-encrypt is not needed in HOOKS. So I'll remove it from the array.

  74. When you done reviewing & modifying config files, select "BACK" & press ENTER.

  75. Finally Select "Done" & press ENTER.

  76. Select "Yes" & exit the installer.


That's it! Archcraft is installed on your system. Reboot the machine & enjoy your new OS.
